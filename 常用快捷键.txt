#  IDEA 常用快捷键

|    按键组合     |       功能描述       |
| :-------------: | :------------------: |
|   alt + a[^*]   |       添加注释       |
|     alt + /     |       补全单词       |
|      psvm       |        主函数        |
|    sout 系列    |      控制台输出      |
| ctr + shift + u |      大小写切换      |
|   ctr + click   |       查看源码       |
|     ctr + b     |     查看方法调用     |
| ctr + shift + i | 视窗查看代码（只读） |
|     ctr + e     |     最近访问文件     |
|   shift + F6    |        重命名        |
|  ctr + alt + u  |       查看类图       |
|                 |                      |
|                 |                      |
|                 |                      |

[^*]:*此快捷键默认无，需要自行设置*
